ex07.html

1. 모든 html 요소(태그)에는 요소 유형에 따라 기본 표시(display) 값이 있다.
2. 표시 값에는 2가지가 있다. (블럭모드와 인라인모드)
- 블럭모드 : 요소 전체 너비를 차지 => 대표적으로 p, div, li, hn, dl, dd, dt, ol, ul, hr, blockquote 태그 등등
- 인라인모드 : 새로운 줄에서 시작X, 필요한 너비 만큼만 차지한다. => 대표적으로 span, img, a, map, b, button, i, q, sub, sup, input 태그 등등

div 태그

1. 웹 페이지의 배치(layout)를 담당하는 태그이다.
2. 블럭모드이다.
3. 일반적으로 다른 요소들을 담을 수 있는 컨테이너의 종류 중에 하나이다.
* 컨테이너 : 다른 요소를 자식으로 가지고 있다.
