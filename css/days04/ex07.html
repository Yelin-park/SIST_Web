ex07.html - css 4가지 결합자

1. 공백 : 직계 + 후손 포함  [ ex)div p{} : div 태그의 직계 자손p + 후손p ]
2. > : 직계 자손만 [ ex)div > p{} : div 태그의 직계 자손p ]
3. + : 태그 바로 뒤에 오는 형제 1명 [ ex) div + p{} : div 태그 바로 뒤에 따라오는 형제 p ]
4. ~ : 태그 요소에서 인접하지 않아도 되는 뒤에 따라오는 모든 형제 태그 [ ex) div ~ p{} : div 태그 바로 뒤에 따라오는 모든 형제 p ]
