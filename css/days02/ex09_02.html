1. text-indent 속성 : 첫 번째 라인(줄)에 들여쓰기
2. letter-spacing 속성 : 텍스트에서 문자와 문자 사이의 간격(공백) 설정(음수값도 설정 가능-겹쳐보임)
3. line-height 속성 : 라인과 라인 사이의 간격(0은 간격이 작아지고 값이 클 수록 간격이 넓어짐)
4. word-spacing 속성 : 단어와 단어 사이의 간격(공백) 설정(음수값도 설정 가능-겹쳐보임)
5. white-space(공백) 속성 : ex09_03.html에서..

6. word-break 속성 : ex09_03.html 복사 -> ex09_05.html에서..
