map(callback(index, element))
	- 현재 일치하는 집합의 각 요소를 함수를 통해 전달하여 반환 값이 포함된 새 jQuery 개체를 생성
	- 반환 값은 배열을 포함하는 jQuery 개체이므로 .get()기본 배열과 함께 작동하기 위해 결과를 호출하는 것이 일반적
	- 사용하는 이유?
		.map()메서드는 요소 컬렉션의 값을 가져오거나 설정하는 데 특히 유용함
	> 특정한 값을 가져오기 위해 사용
