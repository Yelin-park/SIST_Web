jquery method
	1. width() - 요소 너비
	2. innerWidth() - 요소 너비 + 패딩
	3. outerWidth() - 요소 너비 + 패딩 + 보더 (border)
	4. outerWidth(true) - 요소 너비 + 패딩 + 보더 (border) + 마진(margin)
	
	1. height() - 요소 높이
	2. innerHeight() - 요소 높이 + 패딩
	3. outerHeight() - 요소 높이 + 패딩 + 보더(border)
	4. outerHeight(true) - 요소 높이 + 패딩 + 보더(border) + 마진(margin)
